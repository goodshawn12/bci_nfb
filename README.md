# BCI_NFB: Brain-computer interfaces for neurofeedback intervention
This is a Matlab-based toolbox that implements a neurofeedback BCI system. The system includes experiment design,
online signal processing and feature extraction using EEGLAB, and auditory / visual feedback modality using PsychToolbox. 

## Setup
1. Download or clone this repository
1. Download and setup [PsychToolbox](http://psychtoolbox.org/download.html)
1. Open and run `exp_nfb.m` to see the demo
